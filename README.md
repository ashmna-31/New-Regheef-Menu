# Regheef Static QR Menu Landing Page

## What is included
- Mobile-first static menu website
- Arabic / English language selection
- Sticky categories
- Visual card-based menu
- Optimized WebP food and hero images
- Recolored cream logo asset (#FFF3DD)
- Contact, maps, Instagram, TikTok links

## How to use
Open `index.html` locally, or deploy the whole folder to Vercel/Netlify/static hosting.

## QR code
After deployment, generate a static QR code using the final production URL.
Recommended URL format: `https://yourdomain.com/menu` or `https://menu.yourdomain.com`.

## Notes
- No cart, online ordering, payment, accounts, or popups are included.
- Menu data lives in `menu-data.js`.
- Images are inside `assets/images`.
